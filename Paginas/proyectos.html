Una colección de los proyectos más destacados en los que he trabajado, demostrando mis habilidades técnicas y capacidad para resolver problemas complejos.

Furgonite

Desarrollé Furgonite, una aplicación para la gestión de furgones escolares con autenticación de usuarios, seguimiento en tiempo real, y un dashboard administrativo intuitivo. La plataforma permite a padres, conductores y administradores gestionar rutas, usuarios y notificaciones en un entorno seguro y escalable. Construido bajo arquitectura de microservicios.

Pagina Web EnlacesGallyas.com

Desarrollé el sitio web oficial de Enlaces Gallyas, una empresa de telecomunicaciones en Chile. El sitio presenta una estructura clara, moderna y responsiva, orientada a mostrar los servicios de conectividad y cobertura nacional. Se optimizó para rendimiento, visualización móvil y posicionamiento SEO básico.

Fomulario Intuitivo para generar tickets en apolo

Desarrollé una plataforma web intuitiva para que los trabajadores de Gallyas Telecomunicaciones generen tickets internos de manera eficiente. El sistema se conecta directamente a la API REST de GLPI (Apolo) y permite asignar usuarios automáticamente, incluir observadores, y definir entidades específicas. El diseño está optimizado para la experiencia del usuario y se adapta a dispositivos móviles.

Python Django Glpi html Javascript

SacCorreo

Aplicación web desarrollada para Gallyas Telecomunicaciones, orientada a la gestión automática de tickets de soporte. El sistema permite a los clientes crear solicitudes mediante un formulario seguro conectado a la API de GLPI (Apolo), con validaciones, control de envíos y generación de enlaces temporales. Incluye reportes automáticos y un diseño visual atractivo inspirado en estilo Matrix.

Python Django MySQL HTML
